# blend
Web application for marging pdf files.
### Technology:
- Java 20
- Vaadin
- Springboot
### Used library
com.itext (https://mvnrepository.com/artifact/com.itextpdf/itextpdf)
#### Color Palette
- #e38ad9
- #199f97
### Deployment
1. Download latest release.
2. Unpack zip to desired location.
3. Run command:
```bash
java -jar blend-1.0.0.jar
```
